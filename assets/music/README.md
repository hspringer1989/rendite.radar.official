# Hintergrundmusik

Lizenzfreie Musik-Tracks (mp3/wav) hier ablegen — der Renderer wählt zufällig einen aus
und mischt ihn leise (`MUSIC_VOLUME_DB`) unter das Voiceover. Ohne Dateien wird ohne
Musik gerendert.

Quellen für lizenzfreie Tracks (Lizenz beim Download prüfen und hier dokumentieren):
- https://pixabay.com/music/ (Pixabay-Lizenz, auch kommerziell)
- https://incompetech.com/ (CC-BY, Namensnennung in der Caption nötig)

**Wichtig:** Keine Instagram-In-App-Musik oder kommerzielle Songs verwenden — bei
API-Publishing gibt es dafür keine Lizenz.
